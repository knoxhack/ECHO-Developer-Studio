# ECHO Developer Studio

Desktop app for ECHO module, platform, release, and developer workflows.

## Purpose

Desktop app for ECHO module, platform, release, and developer workflows.

## What Lives Here

Studio source, Electron/package configuration, developer workflow docs, release policy docs, and update feed settings.

## Release And Update Role

Owns Developer Studio app releases and update metadata. It consumes SDK schemas and release index metadata.

## Public Or Private

Public is optional. Public helps community developers audit and install the studio; private is reasonable while workflows are internal.

## Build And Dev Commands

Run commands from the repository root.

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run test`

## Artifact Ownership

Developer Studio installers, app update metadata, and studio release notes belong here.
